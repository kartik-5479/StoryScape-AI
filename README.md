# 🌟 Overview

**StoryScape AI** is an AI-powered interactive storytelling application that generates unique visual novel experiences based on user choices.

The application combines **Google Gemini AI**, **Pollinations AI**, and **gTTS** to create engaging adventures where every decision influences the storyline. Users can select different genres and art styles, making every playthrough unique.

# 🚀 Core Functionalities

### 📖 AI Story Generation
Generates engaging story scenes dynamically using Google Gemini AI based on user decisions.

### 🎨 Scene Artwork
Every scene is accompanied by AI-generated artwork to enhance immersion.

### 🔊 Voice Narration
Converts story text into natural speech using Google Text-to-Speech.

### 🎯 Interactive Choices
Every decision affects how the story progresses, creating unique experiences.

### 📚 Story History
Keeps track of previously generated scenes, allowing users to revisit the adventure.

### 🎭 Genre Selection
Supports multiple genres including:

- Fantasy
- Horror
- Mystery
- Adventure
- Sci-Fi

### 🎨 Art Styles

- Anime
- Realistic
- Pixel Art
- Watercolor
- Cyberpunk
